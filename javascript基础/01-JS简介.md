

## JS简介

### JavaScript的组成

JavaScript的三个组成部分:

- **ECMAScript**: JavaScript的**语法标准**

- **BOM**(Broswer Object Model浏览器对象模型),操作浏览器API

- **DOM** (Document Object Model文档对象模型),操作页面元素API
